\doxysubsection{Detailed Description}
sub server for a subtree in the d\+HTT 

This class is intended to be used by GOi\+TRs or other similar constructs in order to communicate with the main server and give direct changes to the subtree. A parent GOi\+TR for instance will use this class as an interface with which it can replan it\textquotesingle{}s subtree.

builds the subtree that serves as the base for this action 

Each goitr represents some high level action. The main idea is for each member of the tree to be left high level until the first activation.
