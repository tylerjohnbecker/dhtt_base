\doxysubsection{Detailed Description}
Base class describing the logic of the nodes. 

This is an interface for how to write the logic of a node. This encompasses both task nodes and behavior nodes. Each node needs describe how to initialize, how to perform the auction, what work to perform, how to parse params, and how to calculate activation\+\_\+potential.

Behaviors should perform some action and then return the success of that action and tasks should propogate the message down and wait for the response (or perform some other action first in special cases).
