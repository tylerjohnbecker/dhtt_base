\doxysubsection{Detailed Description}
Implementation of the OR node for d\+HTTs. 

The OR node will be done after a single one of it\textquotesingle{}s children is done, and therefore it chooses the one with the highest activation potential.

Auction behavior for the OR node. 

If this node has not already run one of it\textquotesingle{}s children then it passes the request of the child with the highest activation potential. If a subtask has already been chosen once then it is the only child which is activated and it\textquotesingle{}s requests are passed up until it is done.
