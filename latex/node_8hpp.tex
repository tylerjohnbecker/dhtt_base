logger functions for dhtt nodes 

Because dhtt nodes do not have their own logger (they are not rclcpp\+::\+Node\textquotesingle{}s) this serves as a wrapper function which allows them to use the stream syntax and utilize the communication aggregator\textquotesingle{}s logger with their own name appended.
