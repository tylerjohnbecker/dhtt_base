\doxysubsection{Detailed Description}
Base class for the plugs which provides the necessary multithreading capabilities. 

This class facilitates the communication between nodes by giving a general interface of methods they can call. For now, all calls are asynchronous and the synchronous calls are obfuscated from the user.
