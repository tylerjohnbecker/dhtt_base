checks given predicate list for internal contradictions 

An internal contradiction in this context involves for instance having the following predicate relationship ( p /\textbackslash{} !p ) or ( p \textbackslash{}/ !p ). If a predicate is meant to realize both possible values we consider that a contradiction for now. However, logically this will not cover every possible contradiction that could occur in predicate relationships as testing if a predicate conjunction is logical is NP-\/\+Hard (SAT).
