\doxysubsection{Detailed Description}
Implementation of the AND node for d\+HTTs. 

The AND node has no constraint on the ordering of the children and so it just runs all of them in decreasing order of activation potential. The AND node also has to run all of it\textquotesingle{}s children.

auction behavior for the AND node. 

An AND node runs all of it\textquotesingle{}s children in an unspecified order. The auction activates all children and collects their requests. Then, the request with the highest activation potential is chosen and sent up the tree. All other children are returned to the WAITING state.
