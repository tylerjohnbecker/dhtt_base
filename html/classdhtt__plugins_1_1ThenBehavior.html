Detailed Description

Implementation of the THEN node for dHTTs.

A THEN node has to run all of it's children in a specified order. The order is currently taken from the literal order that they are listed in the yaml description file.

auction behavior for the THEN subtask

The then node always activates all of the children which haven't finished in order to calculate the activation potential, however, only the request of the next child in the queue is sent up the tree. Passed resources are given immediately to children for consideration when creating their request.
